# Entrega 2 – Organización de Computadores  
## Implementación de la Raíz cuadrada y letras en ASM (Hack)

Este repositorio contiene la implementación de dos posibles algoritmos para calcular la parte entera de la raíz cuadrada y la muestra de cómo se grafican letras en pantalla en ASM (Hack).

---

## Estructura del Repositorio

### **1. `/codigos`**
Contiene los archivos `.hdl` correspondientes a los circuitos implementados:

- `ShiftLeft16.hdl`
- `ShiftLeft64.hdl`

Además, incluye el archivo de pruebas:

- `ShiftLeft16.tst` — utilizado para ejecutar las pruebas del ShiftLeft16 en el Hardware Simulator de Nand2Tetris.

---

## Descripción General del Proyecto

El objetivo de este proyecto fue implementar la operación **Shift Left** para buses de 16 y 64 bits utilizando únicamente puertas lógicas, siguiendo las restricciones del entorno *Nand2Tetris*.  

El desarrollo incluye:

- Diseño y codificación de los componentes `.hdl`
- Creación del netlist estructural  
- Diseño gráfico del circuito en Logisim  
- Archivo `.tst` para la validación del ShiftLeft16  
- Documentación técnica completa del proceso
